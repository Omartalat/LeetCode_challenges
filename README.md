# LeetCode Challenges

A collection of LeetCode problem solutions implemented in JavaScript.

### 1. Roman to Integer
- **File:** [01-roman_to_integer.js](01-roman_to_integer.js)
- **Description:** Convert Roman numerals to integers
- **Difficulty:** Easy

### 2. Palindrome Number
- **File:** [02-palindrome_number.js](02-palindrome_number.js)

## How to Use

Each file contains a standalone solution that can be run with Node.js:

```bash
node 01-roman_to_integer.js
node 02-palindrome_number.js
```
